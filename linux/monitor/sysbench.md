# Lệnh sysbench

https://www.howtoforge.com/how-to-benchmark-your-system-cpu-file-io-mysql-with-sysbench

## Cài đặt

Trên Ubuntu

    apt-get install sysbench -y

Trên Centos

    rpm --import /etc/pki/rpm-gpg/RPM-GPG-KEY*
    yum -y install epel-release
    yum -y update
    yum install sysbench -y
